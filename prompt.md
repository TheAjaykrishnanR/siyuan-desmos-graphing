I want to create a graphing plugin for siyuan-note, a note taking application. The graphing calculator that must be embedded is the popular graphing calculator desmos whose offline version have been created for this very purpose and can be found in the `offline-desmos` folder. You should register a slash command and when i select the desmos graph block, i must get a fully functional embed of this offline desmos page right within siyuan note.

## Subfolders:

- I have put `siyuan-plugin-api-reference` folder which contains siyuan's plugin api which you can use as a reference
- I have added a sample plugin: `plugin-sample` which too can be used as a reference.
- I have put `offline-desmos` folder which contains a fully functional offline version of the graphing calculator desmos. The graphinc calculator can be accessed from desmos.html
