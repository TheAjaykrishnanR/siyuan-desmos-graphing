# Desmos Graphing for SiYuan

Embed Desmos graphing calculators directly into your SiYuan notes.

## Features

- **Graphing Calculator**: Insert a fully functional Desmos calculator via slash command (`/desmos`).
- **State Persistence**: Automatically saves your equations and graph state within the document.
- **Offline Ready**: Works without an internet connection using local assets.

## Usage

1. Type `/desmos`, `/graph`, `/math`, `/数学`, or `/绘图` in any document.
2. Select "Insert Desmos" from the menu.
3. Your graphs are saved automatically as you work.
